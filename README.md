Project Title

Gaming-Academic-Performance

Overview

This project focuses on analyzing student behavioral patterns related to gaming habits, academic performance, stress levels, attendance, sleep schedules, and device usage. The analysis helps identify how gaming addiction and lifestyle factors impact student grades and overall performance.

The project uses Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn for data cleaning, visualization, and exploratory data analysis (EDA).

Dataset Information

The uploaded dataset contains 8000 student records with 14 attributes related to gaming behavior, academic performance, stress levels, attendance, and digital lifestyle analysis.

Features Included
Column Name	Description
student_id	Unique ID of each student
age	Age of the student
gender	Gender of the student
gaming_hours	Daily gaming hours
study_hours	Daily study hours
sleep_hours	Daily sleep duration
attendance	Attendance percentage
gaming_genre	Preferred gaming genre
social_activity	Social interaction hours
device_usage	Daily device usage time
reaction_time_ms	Reaction time in milliseconds
addiction_score	Gaming addiction score
stress_level	Stress level of the student
grades	Academic performance score

Key Insights
Students with higher gaming hours generally showed lower academic grades.
Study hours had a positive relationship with academic performance.
High addiction scores were linked to lower attendance and increased stress.
Proper sleep duration improved focus and reduced stress levels.
Excessive device usage negatively impacted productivity.
Students with balanced gaming and study schedules performed better academically.
